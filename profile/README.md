# The Camus Project

> Misname an object adds to the misfortune of this world.
> 
> Mal nommer un objet, c'est ajouter au malheur de ce monde.
> 
> — Albert Camus, *Sur une philosophie de l'expression*, 1944

The Camus Project provides a method, specification language, programming language and tooling for AI-generated code — code that must be understood, verified, and assumed by humans.

More on [camus-project.github.io](https://camus-project.github.io/)
